0,basic
    a,good*.*
	标准的excel2003/2010文档.
    b,*.xls
	excel使用名为biff的文档格式,文档按照二进制存储.
    c,*.xlsx
	excel开始采用office open xml, 开放标准.
	一份文档数据可以被不同的应用程序解读,数据共享.
    d,*.xml
	office 2003 excel 支持的一种excel文件格式,后缀为xml,但指明了 application
	<?mso-application progid="Excel.Sheet"?>
	also known as SpreadsheetML. Now changed to Office Open XML.

    e,ooxml,office open xml
	NOT the same as Open Office XML
	    or the Open Document Format (ODF) that underlies the OpenOffice.org
	Spreadsheet files are broken into multiple XML files (which are all then zipped)
